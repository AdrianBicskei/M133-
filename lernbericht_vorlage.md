## Beschreibung

Um eine Benutzereingabe w채hrend einer Sitzung im Controller zu speichern, muss man als Erstes beim inputText Formularelement den Value an den Controller weitergeben, wo es als String gespeichert wird. 


Benutzereingabe durch Formularfeld
```
 <h:outputLabel for="eingabe" value="Ihr Name: "/> 
 <h:inputText value="#{helloManagedBean.name}" id="name"/>
```           
            
Speicherung im Controller            
```            
private String name;
            
public String getName() {
  return name;
}

public void setName(String name) {
  this.name = name;
}
```
